Foster Requests Overview

go to Staff > Foster Requests in your management menu to access all foster requests that have been filed. When you do you will be presented with a table that looks similar to the following:

1 Table navigation. How many items to display in the table at any one time.

2 Table organization. These arrows can be found next to each header item. These allow you to reorder the content within that column alphabetically or numerically. Simply click on the arrow to reorder.

3 The status of the current application. There are five different status items which are Waiting, In Review, Approved, Rejected and Closed

4 Table navigation. Simply enter the search parameter you are looking for here. The table will automatically filter out all other results and present everything it finds that matches that parameter.

5 Table navigation. Advances or reverses the next set of of applications.

6 Application selection. Use this dropdown to select the foster application you wish to view. All applications are identified by their unique ID number.

Once you choose an application to view, you will see a window similar to the following, appear below the table.

Application Details

This window is broken down to three major sections. Animal, User and Application.

Animal. This tab gives you a brief overview of the animal that has been applied for. To see all of the animal details, simply click on the animal name. A new window will open with the entire animal profile.

User. This is the user account that has applied for the animal. You'll get a brief overview of the user in question here. To see the entire user account, simply click on the username. A new window will open with the entire user profile.

Application. This is the application that the user has filled out for this foster.

Assign Status. This is a dropdown menu of possible values. The assign status refers only to the foster application you are currently viewing. Once you have assigned a status to the application, press the save button at the bottom of the page to save it.

Notes. This is where you can make any notes on the file. When you save them, your note will be timestamped, saved to the database, then appear below the notes area.
