Add Terms To a Vocabulary

To add additional terms to a vocabulary, do the following:

  • From the Management Navigation Bar, go to Structure > Taxonomy
  • Click the list terms link on the vocabulary name you want to add more terms to. This will display all the current terms in the vocabulary name, and give you the option to add more terms to the list
  • Click the ADD TERM button
  • Add the name of the new term and save

Name. The name of the term you want to add

Description. An administrative description of the term being added

Relations

Parent terms. If the term you are adding is a sub-term of an already existing term, you can set that attribute here. Using the drag-and-drop handles under the list terms link does the same thing. For example: A Vocabulary called "Automobile Manufactures" has the terms "Honda", "Toyota", "Ford", "BMW", etc. These terms could then have sub-terms that are the models each automobile manufacture. Honda would have sub-terms, "Civic", "Accord", "CR-V", etc. Ford could have the terms "Focus", "Mustang", "F-Series Trucks", etc. This principle can also be applied recursively. For example, the sub-term "F-Series Trucks", could have further sub-terms "F-150", "F-250", "F-350", etc.

Weight. Where in the list this term is listed. Higher numbers sink to the bottom.
