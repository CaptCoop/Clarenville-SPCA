Approve and Close a Foster Application Request

If the application was approved, then once the user has picked up their fostered animal. Capture this activity in the notes section and change the Application Status to Closed.

You should then access the animal profile of the fostered animal and remove them from the Foster Program and save. This will also remove the "Foster Me" button from their profile.

If you leave the animal status as "Available for adoption", they will still show on the website as if the animal is still in the shelter.
