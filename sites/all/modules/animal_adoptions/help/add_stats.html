Add a New Statistics File

To add new Stats to the website, go to Content > Add Content > Statistics, then fill out the following fields:

Title. This is the name you give to the file you area about to add.

Body. Enter any spacial details, notes or anything else you like, here.

File. This is where you'll upload your .pdf file of this past years statistics. The only file types that can be added here are txt, pdf, jpeg, png

Click the green SAVE button to save your data.
