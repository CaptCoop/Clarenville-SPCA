Update email addresses in Webforms

From the management navigation menu, go to Content>Webforms

Select the edit link on the webform you wish to update

  • Look along the top, right-hand side of the screen and make sure you have the (1) "Webform" tab selected.
  • Select the (2) "E-Mails" sub-tab and you will display a page similar to the following

On the (3) "Form submission from:" entry, click the (4) edit link to see the following configuration page

Enter/Remove any email address from the E-Mail to address > Custom: line. Each address must be separated by commas. For example: email1@gmail.com,email2@gmail.com,email3@gmail.com

Save your changes
